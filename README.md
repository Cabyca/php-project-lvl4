# Менеджер задач

#### Использование:
На главной странице введите URL проверяемой страницы  
Нажмите "ПРОВЕРИТЬ"  
Посмотрите результат

#### Требования:
PHP (версия ^8.0)  
Composer

#### Установка:
$ https://github.com/Cabyca/php-project-lvl4.git  
$ cd php-project-lvl4  
$ make setup

#### Локальный запуск:
$ make start
